# Ramadia Ranmal Holiday Resort — Wedding Campaign Case Study

**🔗 Live Site:** [View the Interactive Case Study](https://shiru-28.github.io/RAMADIA-WEDDING-CAMPAIGN-CASE-STUDY/)

## Overview
A portfolio case study showcasing a targeted Sinhala-language wedding lead campaign that turned a dead off-season into a fully booked calendar for Ramadia Ranmal Holiday Resort.

## Key Results
- **~1,000** Inbound Messages in just 3 weeks
- **Fully booked** off-season calendar
- **High** lead-to-booking conversion rate
- **Low** cost-per-lead through hyper-local targeting
